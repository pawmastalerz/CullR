# Privacy Policy

## Overview

This application is fully offline and does not collect, store, transmit, or share any personal or sensitive user data.

The app is free, open source, and does not require user accounts, authentication, or network access.

## Media Access

The application requests access to photos and videos on the user's device solely to enable its core functionality.

- Access to media files is performed **locally on the device**
- No media files are uploaded, transmitted, or shared
- No data leaves the user's device at any time

Requested permissions:
- android.permission.READ_MEDIA_IMAGES
- android.permission.READ_MEDIA_VIDEO
- android.permission.READ_EXTERNAL_STORAGE (legacy, for compatibility)

## Data Collection

This application:
- Does **not** collect personal data
- Does **not** collect sensitive user data
- Does **not** use analytics
- Does **not** use advertising
- Does **not** include third-party SDKs that collect data

## Data Sharing

No user data is shared with any third parties.

## Data Retention and Deletion

Since no user data is collected or stored, there is no data retention or deletion process.

## Children's Privacy

This app does not target children and does not knowingly collect any data from children.

## Changes

If this privacy policy changes, updates will be published on this page.

## Contact

For questions regarding privacy, please open an issue in the project repository.
